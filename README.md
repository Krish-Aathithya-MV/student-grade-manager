# Student Grade Manager

A Python-based Student Grade Manager that demonstrates Object-Oriented Programming (OOP), file persistence, and database integration using SQLite. The project is designed to evolve from a simple terminal application into a full-stack student management system.

## Features

- Add Students
- View All Students
- Add Single or Multiple Marks
- Generate Student Report Cards
- Calculate Average Scores
- Assign Letter Grades (A+, A, B, C, D, F)
- Persistent Data Storage using SQLite
- Input Validation and Error Handling

## Technologies Used

- Python
- Object-Oriented Programming (OOP)
- SQLite
- Git & GitHub

## How to Run

1. Clone the repository:

bash git clone https://github.com/Krish-Aathithya-MV/student-grade-manager.git 

2. Navigate to the project folder:

bash cd student-grade-manager 

3. Run the application:

bash python main.py
