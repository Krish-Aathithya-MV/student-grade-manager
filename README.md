# Student Grade Manager

A Python-based Student Grade Manager designed to evolve with databases and web technologies.

## Features
- Add Students
- View Students
- Add Marks
- Generate Report Cards
- Calculate Average Grades
- Assign Letter Grades

## Technologies Used
- Python
- Object-Oriented Programming (OOP)
- Git & GitHub

## Future Roadmap
- V2: JSON File Storage
- V3: SQLite Database Integration
- V4: Web Interface
- V5: AI Features

## How to Run

bash python main.py
